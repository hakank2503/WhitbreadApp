# Foursquare Integration with Spring Boot MVC for Whitbread

## Technology Stack
- Java
- Spring Boot
- Spring MVC
- JUnit, Mockito
- JSP, HTML5, CSS3, JavaScript

## What you'll need
- JDK 1.8 or later
- Maven 3 or later


## Maven Build
'mvn clean install'

## Run
'mvn spring-boot:run'


Open the default url: http://localhost:8080/
